# Documentation Assets

This folder contains screenshots, diagrams, and other visual assets for the project.

## Adding Screenshots

1. Take screenshots of your application
2. Save them in this `docs/` folder with descriptive names:
   - `screenshot-homepage.png`
   - `screenshot-substitution-result.png`
   - `screenshot-nutrition-comparison.png`
   - etc.

3. Reference them in the main README.md using markdown syntax:
   ```markdown
   ![Description of image](docs/screenshot-filename.png)
   ```

## Recommended Screenshots

- Homepage/main interface
- Example substitution result
- Nutrition comparison card
- API documentation page (FastAPI /docs)
- Database schema (optional)
